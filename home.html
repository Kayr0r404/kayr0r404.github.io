Full-stack e-commerce website designed for seamless online shopping. It features a React frontend and a Flask backend, ensuring a dynamic user experience with efficient data handling. The key functionalities and technologies used: ...

Frontend (React)

  • Routing: Implemented using React Router to navigate between product listings, product details, cart, and user profiles.
  • State Management: Uses React state to manage UI elements and localStorage for cart and wishlist persistence.
  • User Authentication: Integrated with JWT to allow users to register, log in, and manage their accounts.
  • Wishlist Feature: Users can add/remove products from their wishlist, with a dedicated wishlist page to view saved items.
  • Cart & Checkout: Items can be added to the cart, quantities adjusted, and orders placed seamlessly.
  • Payment Gateway: Integrated with Stripe to process secure payments.

Backend (Flask)

  • User Authentication: Uses JWT for secure login and session management.
  • Database Management: Implemented with SQLAlchemy for handling user data, products, orders, and carts.
  • Product Management: Allows admins to add, update, and delete products.
  • Order Processing: Stores order history, calculates totals, and updates product stock accordingly.

Other Features

  • Responsive Design: Works across mobile, tablet, and desktop devices.
  • ES6 & Babel Support: Ensures compatibility across browsers.
